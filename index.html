Projects

DIY Multi-Functional Robot Car Using Arduino

Built a multi-functional robot car capable of line following, obstacle avoidance, and object manipulation using an Arduino UNO, motor drivers, IR and ultrasonic sensors, and servo motors. The system was programmed and tested for real-time response and mobility, showcasing hands-on skills in embedded systems, robotics integration, and automation control.

Classifying Speech Disorders Using Voice Signals and Machine Learning

Developed a machine learning system that classifies speech disorders such as stuttering, dysarthria, and vocal tremor using voice signal analysis. The project combined synthetic and real voice data, extracted key acoustic features, and applied SVM, Random Forest, and Gradient Boosting models for classification. Results showed strong diagnostic potential, improving accuracy, early detection, and accessibility for speech disorder assessment through AI-driven analysis.

FPGA vs. ASIC in Satellite Embedded Systems with Power Subsystems and Sensor Multiplexing.

A hybrid FPGA-ASIC satellite power subsystem designed to improve energy efficiency and real-time adaptability. The FPGA manages dynamic power control and sensor data, while the ASIC ensures stable voltage regulation. Simulated with VHDL and Midsim, the system proved reliable and efficient for space applications.

I-V Curve Analysis for Solar Cells

Developed a mathematical model and simulation to analyze the current-voltage (I-V) characteristics of photovoltaic (PV) cells under varying temperature and irradiance conditions. Used numerical methods (Newton-Raphson) and visualized the results using Gnuplot, providing insight into solar cell efficiency and performance behavior at different light levels.
