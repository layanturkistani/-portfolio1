  • Tech stack: Python, pandas, matplotlib
  • Skills developed: Data cleaning, filtering, EDA, and visualization

Lab 2 — Data Science Workflow

This lab focused on applying the complete data science pipeline — from data ingestion and transformation to model training and performance evaluation. I implemented a simple machine learning model using scikit-learn, applied feature encoding and normalization, and analyzed prediction accuracy through metrics and graphs. The experience enhanced my understanding of how raw data evolves into actionable insights through AI-driven modeling.

  • Tech stack: Python, scikit-learn, pandas, seaborn

The third lab combined all prior skills to build a predictive model that classifies weather conditions based on temperature, humidity, and wind speed data. I trained multiple classifiers, including Decision Trees and Random Forests, and compared their accuracy through confusion matrices and visualizations. Beyond coding, the focus was on interpreting model outputs and understanding how data features influence predictions in real-life applications such as climate forecasting and smart agriculture.

  • Tech stack: Python, scikit-learn, matplotlib
  • Skills developed: Model comparison, data visualization, feature importance analysis
